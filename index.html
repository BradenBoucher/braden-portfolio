ColorCoded Music

iOS App · Available on the App Store
Website App Store

Overview

ColorCoded Music is an iOS application that transforms the way beginner and intermediate pianists learn to read sheet music. Using on-device machine learning, the app scans any PDF of sheet music, detects individual noteheads, and overlays a color-coded system directly onto the page — making it immediately clear which notes to play and where to find them on the keyboard.

The app also generates intelligent fingering suggestions for each hand, helping players build proper technique from the start. Whether you're working through a classical piece, a pop arrangement, or a hymnal, ColorCoded gives you a visual shortcut to understanding the music.

Screenshots

How It Works

At the core of ColorCoded is a custom-trained YOLOX object detection model running entirely on-device through Apple's CoreML framework. The model was trained on thousands of annotated sheet music samples across 24 notehead classes, achieving high accuracy at 1280×1280 input resolution. When a user imports a PDF, the app processes each page through the following pipeline:

  • The PDF is rendered at high resolution and passed through the YOLOX CoreML model to detect noteheads with bounding box coordinates and class predictions.
  • Detected noteheads are mapped to musical pitch using staff-relative positioning. The system identifies staff lines, assigns notes to the correct hand (right hand for even-indexed staves, left for odd), and normalizes pitch values.
  • Each pitch class receives a distinct color — C is red, D is orange, E is yellow, F is green, G is cyan, A is blue, and B is violet — creating an intuitive visual rainbow across the musical scale.
  • A rule-based fingering engine (FingeringSuggester v6.4) runs a six-pass pipeline to assign fingers 1–5, accounting for hand position, chord spacing, phrase boundaries, and scale patterns.
  • The colored overlays and fingering numbers are rendered directly onto the page, which the user can zoom, scroll, and navigate seamlessly.

Key Features

  • PDF sheet music import with multi-page support and page range selection
  • On-device ML notehead detection — no internet connection required, fully private
  • Color-coded pitch visualization using a 7-color chromatic system
  • Automatic piano fingering suggestions for both hands
  • Public library for sharing annotated scores with other users
  • Scan confirmation workflow with start/end page selection and append mode
  • Built with Swift, SwiftUI, CoreML, and Firebase

Workspace MVP

Web Application · Next.js
Live Demo

Overview

Workspace MVP is an AI-powered productivity platform that combines the flexibility of a freeform canvas with the structure of spreadsheets and the intelligence of natural language AI. It's designed for individuals and small teams who need a single environment to organize information, run calculations, and take action — all through conversation.

Rather than switching between a notes app, a spreadsheet, and a chat interface, Workspace brings everything into one unified space where blocks of content live on an infinite canvas and can be manipulated by AI or by hand.

Key Features

  • Infinite canvas with draggable, resizable content blocks (CanvasBlock architecture)
  • Integrated AI spreadsheet — read and write cell data, format columns and rows through natural language
  • Action confirmation UI with undo support for AI-triggered operations
  • Full-text search across all blocks with ⌘+F keyboard shortcut
  • Real-time sync with Firebase Firestore for persistence and collaboration
  • Deployed on both Vercel and Railway for flexibility and longer timeout support

History Legends

Web Game · React
Play Game

Overview

History Legends is an educational RPG-style quiz game designed to make studying AP United States History engaging and effective. The game covers all 7 APUSH eras with over 345 carefully balanced questions, organized across 40 nodes on a progression map. Players advance through historical eras, earn experience, and unlock new content as they master the material.

The game features custom sprite art for each era, generated through a combination of Leonardo.ai for image-to-image processing and Gemini for text-to-image generation, followed by a Python cleanup pipeline for background removal and consistent sizing.

Screenshots

Key Features

  • 345+ carefully balanced quiz questions across 7 APUSH eras with answer length ratios between 1.2–1.8x to prevent guessing by length
  • 40-node progression map with branching paths and era-specific challenges
  • Custom sprite art pipeline: AI generation → Python cleanup (flood-fill background removal, resize to 512px height)
  • RPG-style experience system with leveling, unlockable content, and era completion tracking
  • Responsive design that works on desktop and mobile browsers
  • Deployed on Firebase Hosting with Git-based CI/CD workflow

StockFlow

Cross-Platform App · React Native
Live Demo

Overview

StockFlow is a cross-platform inventory management application built for construction companies and field teams who need to track materials, tools, and supplies across job sites. The app runs on both iOS and web, giving office managers and on-site workers a shared real-time view of inventory levels, movements, and usage history.

Designed around the daily workflows of construction operations, StockFlow replaces clipboard-and-spreadsheet tracking with barcode scanning, real-time sync, and role-based access — reducing errors, preventing stockouts, and saving hours of manual data entry every week.
